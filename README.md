# Gaming Organization Data Analysis

## Overview

This project is a sample data analysis project based on a gaming organization. Some data is inspired by real-world scenarios, while some records are randomly generated for learning and practice purposes.

## Tools Used

* Microsoft Excel 2019
* PostgreSQL (pgAdmin)

## Project Features

* Designed a relational database
* Imported data from Excel into PostgreSQL
* Performed data analysis using SQL queries
* Used JOINs, GROUP BY, HAVING, ORDER BY, Aggregate Functions, and CASE statements

## Database Tables

* Creators
* Brands
* Esports Teams
* Lineups
* Backend Employees

## Purpose

This project was created to practice SQL database design, data management, and data analysis skills as part of a Data Analyst portfolio.
